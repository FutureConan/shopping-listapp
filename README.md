# 쇼핑 리스트 앱

순수 HTML/CSS/JavaScript로 만든 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료 처리 (취소선 적용)
- 개별 항목 삭제
- 완료된 항목 일괄 삭제
- 완료 현황 요약 표시
- localStorage로 데이터 영속 저장

## 사용 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML / CSS / JavaScript (바닐라)
- Google Fonts (Noto Sans KR)
- localStorage API
